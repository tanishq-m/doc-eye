# Campaign Performance Dashboard Guide

## Dashboard Overview
The Campaign Performance Dashboard provides real-time metrics across all active campaigns.

## Key Metrics
- Impressions
- Clicks
- Conversions
- Cost per Acquisition (CPA)
- Return on Ad Spend (ROAS)

## Dashboard Sections
1. **Overview** - High-level KPI summary
2. **Channel Performance** - Breakdown by channel
3. **Campaign Details** - Individual campaign metrics
4. **Audience Insights** - Demographics and interests
5. **Trends** - Historical performance

## Accessing the Dashboard
1. Log into platform
2. Navigate to Reports > Performance Dashboard
3. Select date range
4. Filter by campaigns or channels

## Exporting Reports
- Export as PDF for stakeholder reports
- Export as CSV for analysis
- Schedule automated weekly reports
